# fate-draw-app

「命运之签」前端应用，基于 Vue 3 + Vite + TypeScript + Tailwind CSS。

完整说明（本地开发、Cloudflare Pages 部署）见仓库根目录 [README.md](../README.md)。

## 快速开始

```bash
npm install
npm run dev
```

## 常用命令

| 命令 | 说明 |
|------|------|
| `npm run dev` | 启动开发服务器 |
| `npm run build` | 生产构建，输出到 `dist/` |
| `npm run preview` | 预览构建结果 |
